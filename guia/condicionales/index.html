    Condicionales en Java

    Las estructuras condicionales permiten que un programa tome decisiones y ejecute diferentes bloques de código según una condición.

    ¿Por qué usarlos?

    Sin condicionales, todos los programas ejecutarían siempre las mismas instrucciones. Los condicionales permiten adaptar el comportamiento según los datos.

    Ejemplo real

    Un cajero automático verifica si tienes saldo suficiente antes de permitir un retiro. Eso es un condicional.

    Tipos de condicionales

    • if — ejecuta un bloque si la condición es verdadera
    • if - else — ejecuta uno u otro bloque según la condición
    • if - else if - else — evalúa múltiples condiciones
    • switch — compara un valor contra varios casos

    if

    int edad = 20;

if (edad >= 18) {
    System.out.println("Eres mayor de edad");
}

    Funcionamiento

    El bloque dentro de {} solo se ejecuta si la condición entre () es true.

    if - else

    int edad = 15;

if (edad >= 18) {
    System.out.println("Puedes votar");
} else {
    System.out.println("Aún no puedes votar");
}

    if - else if - else

    Se usa cuando hay más de dos posibilidades:

    int nota = 75;

if (nota >= 90) {
    System.out.println("Sobresaliente");
} else if (nota >= 70) {
    System.out.println("Notable");
} else if (nota >= 50) {
    System.out.println("Aprobado");
} else {
    System.out.println("Suspenso");
}

    Orden importa

    Las condiciones se evalúan de arriba a abajo. En cuanto una se cumple, las demás se omiten.

    switch

    Se usa cuando comparamos una variable contra valores fijos:

    int dia = 3;

switch (dia) {
    case 1:
        System.out.println("Lunes");
        break;
    case 2:
        System.out.println("Martes");
        break;
    case 3:
        System.out.println("Miércoles");
        break;
    case 4:
        System.out.println("Jueves");
        break;
    case 5:
        System.out.println("Viernes");
        break;
    default:
        System.out.println("Fin de semana");
}

    No olvides el break

    Sin break, el programa seguirá ejecutando los siguientes case aunque no coincidan. A esto se le llama fall-through.

    Operadores de comparación

    Operador Significado Ejemplo
    == Igual a edad == 18
    != Distinto de estado != "activo"
    > Mayor que saldo > 0
    < Menor que temperatura < 0
    >= Mayor o igual nota >= 50
    <= Menor o igual edad <= 65

    Error común: == vs =

    • = asigna un valor: edad = 18
    • == compara dos valores: edad == 18

    Usar = dentro de un if produce un error de compilación en Java.

    Operadores lógicos

    Permiten combinar varias condiciones:

    Operador Nombre Resultado
    && AND true solo si ambas son true
    \|\| OR true si al menos una es true
    ! NOT Invierte el valor: !true = false 
    int edad = 25;
boolean tieneCarnet = true;

// AND: debe cumplir ambas
if (edad >= 18 && tieneCarnet) {
    System.out.println("Puede conducir");
}

// OR: basta con una
if (edad < 12 || edad > 65) {
    System.out.println("Tarifa especial");
}

// NOT: niega la condición
boolean bloqueado = false;
if (!bloqueado) {
    System.out.println("Acceso permitido");
}

    Operador ternario

    Una forma compacta de escribir un if - else simple:

    // Sintaxis: condición ? valorSiTrue : valorSiFalse
int edad = 20;
String resultado = (edad >= 18) ? "Mayor de edad" : "Menor de edad";
System.out.println(resultado); // Mayor de edad

    ¿Cuándo usarlo?

    Úsalo solo para casos simples. Si la lógica es compleja, es más claro usar un if - else normal.

    Ejemplo completo

    Main.java
    public class Main {
    public static void main(String[] args) {

        int nota = 65;
        boolean haEntregado = true;

        if (!haEntregado) {
            System.out.println("Suspenso por no entregar");
        } else if (nota >= 90) {
            System.out.println("Sobresaliente");
        } else if (nota >= 70) {
            System.out.println("Notable");
        } else if (nota >= 50) {
            System.out.println("Aprobado");
        } else {
            System.out.println("Suspenso");
        }
    }
}

    Aprobado

    Conclusión

    Resumen

    • if ejecuta código si una condición es true
    • else y else if añaden alternativas
    • switch es ideal para comparar un valor contra casos fijos
    • Los operadores &&, || y ! permiten combinar condiciones
