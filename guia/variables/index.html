    Variables en Java

    Una variable es un espacio en memoria que almacena un valor que puede cambiar durante la ejecución del programa.

    Declaración de variables

    La sintaxis básica para declarar una variable en Java es:

    tipo nombreVariable = valor;

    Convención de nombres

    En Java se usa camelCase para nombrar variables: la primera palabra en minúscula y las siguientes con mayúscula inicial. Ejemplo: nombreCompleto, edadUsuario.

    Tipos de datos primitivos

    Java tiene 8 tipos primitivos. Estos son los más usados:

    Tipo Tamaño Descripción Ejemplo
    int 32 bits Números enteros int edad = 20;
    double 64 bits Números decimales double pi = 3.14;
    float 32 bits Decimales (menos precisión) float x = 1.5f;
    long 64 bits Enteros muy grandes long n = 100000L;
    char 16 bits Un solo carácter char letra = 'A';
    boolean 1 bit Verdadero o falso boolean ok = true;
    byte 8 bits Números pequeños (-128..127) byte b = 100;
    short 16 bits Enteros cortos short s = 32000;

    Tipo String (objeto)

    String no es un tipo primitivo, sino un objeto. Se usa para almacenar texto.

    String nombre = "María";
String saludo = "Hola, " + nombre; // Concatenación

    String vs char

    • char almacena un solo carácter entre comillas simples: 'A'
    • String almacena texto completo entre comillas dobles: "Hola"

    Ejemplo completo

    Main.java
    public class Main {
    public static void main(String[] args) {

        int edad = 20;
        double altura = 1.75;
        char inicial = 'M';
        boolean esMayor = true;
        String nombre = "María";

        System.out.println("Nombre: " + nombre);
        System.out.println("Edad: " + edad);
        System.out.println("Altura: " + altura + " m");
        System.out.println("Inicial: " + inicial);
        System.out.println("Mayor de edad: " + esMayor);
    }
}

    Salida esperada

    Nombre: María
Edad: 20
Altura: 1.75 m
Inicial: M
Mayor de edad: true

    Constantes

    Si una variable no debe cambiar, se declara con la palabra clave final:

    final double PI = 3.14159;
final int MAX_INTENTOS = 3;

    Convención

    Las constantes se escriben en MAYÚSCULAS con guiones bajos: MAX_VELOCIDAD, PRECIO_BASE.

    Modificar el valor de una variable

    El valor de una variable puede cambiar en cualquier momento (excepto si es final):

    int puntos = 0;
System.out.println(puntos); // 0

puntos = 100;
System.out.println(puntos); // 100

puntos = puntos + 50;       // También: puntos += 50;
System.out.println(puntos); // 150

    Operadores aritméticos

    Operador Operación Ejemplo
    + Suma 5 + 3 = 8
    - Resta 5 - 3 = 2
    * Multiplicación 5 * 3 = 15
    / División 10 / 2 = 5
    % Módulo (resto) 10 % 3 = 1 
    int a = 10, b = 3;

System.out.println(a + b);  // 13
System.out.println(a - b);  // 7
System.out.println(a * b);  // 30
System.out.println(a / b);  // 3  (división entera)
System.out.println(a % b);  // 1  (resto)

    División entera

    Al dividir dos int, el resultado se trunca: 10 / 3 = 3, no 3.33. Para obtener decimales, usa double: (double) 10 / 3 = 3.33.

    Reglas para nombrar variables

    No se permite

    • Empezar con un número: ~~int 1nombre~~
    • Usar espacios: ~~int mi variable~~
    • Usar palabras reservadas: ~~int for~~, ~~int class~~

    Se recomienda

    • Nombres descriptivos: edadUsuario mejor que x
    • camelCase: precioProducto, nombreCompleto
    • Evitar abreviaturas confusas: temperatura mejor que tmp

    Primitivos vs Objetos

    Almacenan el valor directamente en memoria.

    int a = 5;
int b = a;   // b es una copia independiente
b = 10;
System.out.println(a); // 5 (no cambia)

    Almacenan una referencia a la dirección de memoria.

    String s1 = "Hola";
String s2 = s1;
// s1 y s2 apuntan al mismo objeto en memoria

    Conclusión

    Resumen

    • Las variables almacenan datos en memoria con un tipo definido
    • Java tiene 8 tipos primitivos y el tipo objeto String
    • Usa final para declarar constantes
    • Sigue la convención camelCase para nombrar variables
