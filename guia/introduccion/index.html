    Introducción a Java

    Java es uno de los lenguajes de programación más populares del mundo, conocido por su portabilidad, seguridad y robustez.

    Historia de Java

    Java fue creado por James Gosling en Sun Microsystems en 1995. Su diseño original se llamaba Oak y estaba pensado para electrodomésticos inteligentes. Hoy en día es propiedad de Oracle y sigue siendo uno de los lenguajes más usados a nivel global.

    Curiosidad

    Java fue diseñado con el lema "Write Once, Run Anywhere", es decir, escribe el código una vez y ejecútalo en cualquier plataforma.

    Características principales

    Característica Descripción
    Multiplataforma Funciona en cualquier SO con la JVM instalada
    Orientado a objetos Todo se basa en clases y objetos
    Tipado estático Los tipos de datos se declaran antes de usarse
    Robusto Gestión automática de memoria (Garbage Collector)
    Seguro Sandbox de seguridad integrado en la JVM

    ¿Cómo funciona Java?

    Java no se compila directamente a código máquina. En su lugar, sigue este proceso:

    Código fuente (.java)
        ↓
   Compilador (javac)
        ↓
   Bytecode (.class)
        ↓
JVM (Java Virtual Machine)
        ↓
   Ejecución del programa

    ¿Qué es la JVM?

    La Java Virtual Machine (JVM) es el motor que interpreta el bytecode y lo ejecuta en tu sistema operativo. Gracias a ella, el mismo programa funciona en Windows, Linux y macOS sin cambiar nada.

    Compilación y ejecución

    Para ejecutar un programa Java desde la terminal:

    javac Main.java   # Paso 1: compilar (genera Main.class)
java Main         # Paso 2: ejecutar

    Importante

    El nombre del archivo .java debe coincidir exactamente con el nombre de la clase pública. Si la clase se llama Main, el archivo debe llamarse Main.java.

    Tu primer programa en Java

    Main.java
    public class Main {
    public static void main(String[] args) {
        System.out.println("¡Hola, mundo!"); // (1)
    }
}
    1. System.out.println imprime un texto en la consola y añade un salto de línea al final.

    Explicación línea por línea

    Fragmento Significado
    public class Main Define una clase pública llamada Main
    public static void main(...) Punto de entrada del programa
    String[] args Parámetros que se pueden pasar al ejecutar
    System.out.println("...") Imprime texto en la consola

    Estructura básica de un programa

    Estructura.java
    public class NombreClase {          // (1)
    public static void main(String[] args) { // (2)
        // Aquí va tu código
    }
}
    1. Toda clase pública debe tener el mismo nombre que el archivo.
    2. Todo programa Java empieza ejecutándose desde el método main.

    Conclusión

    Resumen

    • Java fue creado en 1995 y sigue siendo muy relevante hoy
    • El código Java se compila a bytecode y lo ejecuta la JVM
    • Todo programa Java tiene un método main como punto de entrada
    • El nombre del archivo debe coincidir con el nombre de la clase
